- "browser-image-compression" (이미지 압축)

- "react-router-dom" (페이지 생성)

- "react-countup"(카운트 업 효과 적용)
- "react-paginate" (페이지네이션 효과)
-  "crypto-js" (패스워드 암호화 및 복호화)
-  "swiper" (이미지 슬라이더)
-  "typewriter-effect" (타이핑 효과)
